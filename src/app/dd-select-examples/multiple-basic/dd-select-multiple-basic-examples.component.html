Multiple select

This is a multiple select example with dialog. Dialog shows up if you click on select field. In the dialog we present a list component where the user can select multiple items.

In this example we have an address model with a tags field and a set of tags. The user can select one on more tags.

The dialogSettings contains a dialog configuration to show TagListComponent when shows the dialog.

Usage:

    <dd-select
      mode="multiple"
      [model]="address"
      field="tags"
      [items]="tags"
      [dialogSettings]="dialogSettings"
    ></dd-select>

Example:

Model:

{{ address | clearModel | json }}
